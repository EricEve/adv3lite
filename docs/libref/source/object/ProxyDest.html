Details

ProxyDest is a mix-in class for use with an object that represents the exterior of a room, such as a hut in a field. Its purpose is to remove such an object as the possible target of a GO TO command provided there's another alternative. This prevents a GO TO command from taking the player character from the appropriate room to an inappropriate destination.
