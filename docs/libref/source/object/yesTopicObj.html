Singleton object used to trigger a YesTopic; we must make it familiar so that YesTopics can be listed as suggested topics. We define this as noTopicObj in actions.t rather than actor.t so that the SayYes and SayNo actions in actions.t will compile even if actor.t is absent from the build.
