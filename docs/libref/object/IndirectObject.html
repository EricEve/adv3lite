Details

The IndirectObject role is the role of a secondary object that is used in the command, but isn't the primary object being acted upon. This is usually a tool (UNLOCK dobj WITH iobj), a destination (PUT dobj IN iobj), or a topic (ASK dobj ABOUT iobj).
