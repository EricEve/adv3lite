Details

A Scenery object we want to act like a MultiLoc, that is one that creates a series of MultiLoc Scenery objects. Note that a MultiLccScenery object is *not* itself a MultiLoc, so cannot be defined as MultiLoc, Scenery. Rather it is an object that creates a set of MultiLoc Scenery objects (of the MultiScenItem class), which will take their locations from our location properties.

isInitiallyIn (obj)scenery.t[318]
Test an object for inclusion in our initial location list. By default, we'll simply return true to include every object. We return true by default so that an instance can merely specify a value for initialLocationClass in order to place this object in every instance of the given class. The MultiLoc objects we create will use our version of this method.
