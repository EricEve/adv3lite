Details

The DirectObject role is the role of the object being most directly acted upon in the command. The is the only role in a verb that has only one object. In a verb with two objects, this is the object most directly affected. For example, UNLOCK DOOR WITH KEY directly acts upon the door, so the door is the direct object; the key isn't the direct object because it's merely a tool used to effect the change on the door.
