**********************************************************************************

This module provides the LookDir extension for the adv3Lite library (c) 2024 Eric Eve

Version 1.0 09-Dec-2024

The LookDir extension provides handling of a new LookDir action, which handles cdmmands of the form LOOK <DIR>, e.g., LOOK NORTH.

To use it, define xxxLook properties on each room where you want to provide a view in the xxx direction, for example:

northLook = "To the north you see a long row of pines marching into the middle distance. "
