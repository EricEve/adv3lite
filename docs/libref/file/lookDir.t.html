This module provides the LookDir extension for the adv3Lite library (c) 2024 Eric Eve

Version 2.1 24-Jan-25

The LookDir extension provides better handling the LookDir action, which handles cdmmands of the form LOOK <DIR>, e.g., LOOK NORTH.

To use it, define xxxLook properties on each room where you want to provide a view in the xxx direction, for example:

northLook = "To the north you see a long row of pines marching into the middle distance. "

Where the appropriate xxxLook property hasn't been defined, this extension will attempt to generate a description of the direction being looked in based on what is defined on the corresponding direction property of the actor's location.
