adv3Lite Library - module ID's

This module defines a framework for "module ID's." The module ID mechanism allows library modules to identify themselves. The module ID framework is modular, in that several libraries can be linked together without any prior knowledge of one another, and the module ID framework will be able to find all of their ID's.
